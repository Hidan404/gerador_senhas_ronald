# Gerador de Senhas

Um aplicativo Python simples para gerar senhas seguras.

## Funcionalidades

- Geração de senhas aleatórias
- Personalização de comprimento da senha
- Opções para incluir:
    - Letras maiúsculas
    - Letras minúsculas
    - Números
    - Caracteres especiais

## Requisitos

- Python 3.x

## Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/gerador_senhas_ronald.git
```

2. Execute o script:
```bash
python gerador_senhas.py
```
